## Fundamentos

- **Círculo cromático RYB** (Newton, Goethe, Itten) — el modelo que aplica a textiles. 3 primarios
  (rojo, amarillo, azul), 3 secundarios (verde, naranja, violeta), 6 terciarios.
- **Propiedades**: matiz (*hue*), saturación (*chroma*), valor (*value*). Añadir blanco = *tint*;
  negro = *shade*; gris = *tone*.
- **Albers** (*Interaction of Color*, 1963): un color se percibe diferente según los colores que lo
  rodean. **Evalúa cada prenda siempre en contexto del outfit completo.**
- **Cálidos vs fríos**: cálidos (rojo, naranja, amarillo, tonos tierra) "avanzan"; fríos (azul,
  verde-azulado, violeta) "retroceden".

## Neutros

**Clásicos**: negro, blanco, gris, beige, marrón, marino.

**Modernos clave para *quiet luxury***: camel, taupe, greige, crema, marfil, hueso, antracita,
chocolate, verde oliva (funciona como neutro).

> **Regla práctica:** combina preferentemente neutros de la misma temperatura. Si combinas blanco
> puro (frío) con camel (cálido), introduce un puente (marino, gris medio).

## Esquemas de combinación

| Esquema | Definición | Ejemplo |
|---|---|---|
| **Monocromático** | Un matiz en distintos valores y saturaciones | Blazer marino + blusa azul francia + pantalón marino + zapatos marino |
| **Análogo** | 3 colores adyacentes en el círculo | Blusa verde salvia + falda verde bosque + zapatos azul petróleo |
| **Complementario** (suavizado) | Opuestos en versión desaturada | Vestido marino + cinturón camel/cobre + aretes oro |
| **Complementario dividido** | Un color + los 2 vecinos del opuesto | Pantalón marino + blusa crema-camel + zapatos terracota apagado |
| **Triádico** | 3 equidistantes (en versión tierra) | Falda oliva + blusa teja + accesorios ciruela |
| **Tetrádico** | 2 pares complementarios | Pantalón marino + blusa crema + blazer oliva + accesorios burdeos |
| **Esquema neutro** | Combinación de varios neutros | Pantalón camel + camisa seda crema + blazer chocolate + mocasín marrón |

## La regla 60-30-10

- **60 %** color dominante (suele ser neutro): pantalón, falda, vestido base, abrigo.
- **30 %** color secundario: blusa, jersey, blazer.
- **10 %** acento: pañuelo, zapatos, joyería, labial.

## Combinaciones infalibles

- Marino + camel.
- Blanco + negro.
- Camel + crema + blanco (*quiet luxury*).
- Gris + burgundy.
- Marino + blanco + rojo (clásico náutico).
- Negro + blanco + un color.
- Greige + crema + chocolate (paleta tierra contemporánea).
- Beige + verde oliva + marrón (naturaleza sin caer en cliché).

## Combinación de estampados

- **Paleta común** entre estampados.
- **Variar escala** — uno grande + uno pequeño.
- **Rayas y animal print** funcionan casi como neutros.
- **Sólido como amortiguador.**

> En contextos formales, **máximo un estampado por outfit**. En casual elevado pueden mezclarse
> siguiendo estas reglas.

## Psicología del color

| Color | Mensaje | Cuándo usar |
|---|---|---|
| **Marino** | Autoridad, confianza, calma | Por defecto en formal |
| **Negro** | Poder, formalidad máxima | Eventos nocturnos; evitar en días largos colaborativos |
| **Gris** | Profesionalismo neutro | Días técnicos, presentaciones |
| **Blanco / marfil** | Claridad, paz, neutralidad | Como base; cuidado con flashes |
| **Camel / beige** | *Quiet luxury*, cercanía refinada | Reuniones, eventos diurnos |
| **Rojo** | Poder, atención | Cuando quieras visibilidad; no en conversaciones consensuales |
| **Burdeos** | Autoridad cálida sin agresividad | Alternativa al rojo |
| **Verde bosque / botella** | Sostenibilidad, *gravitas* | Tu "marino" alternativo |
| **Verde esmeralda** | Lujo, riqueza | *Statement piece*, cócteles |
| **Verde oliva / musgo** | Estabilidad, naturalismo refinado | Casual elevado |
| **Verde salvia** | Calma, *mindfulness* | Reuniones largas, primavera-verano |
| **Azul medio** | Confianza, comunicación | Entrevistas, mediación |
| **Ciruela / púrpura** | Creatividad, dignidad | *Statement* sutil |
| **Mostaza / terracota** | Calidez tierra, sofisticación | Otoño, acentos |
